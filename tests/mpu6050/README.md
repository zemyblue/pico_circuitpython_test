# MPU6050 test

## Connected Pins
MPU6050 use I2C interface.
```
VIN - 3V3
GND - GND
SCL - GP17
SDA - GP16
```

## Copy adafruit library
Copy below libraries of [circuitpython library](https://circuitpython.org/libraries) to `CIRCUITPY/lib/`
* `lib/adafruit_mpu6050.mpy`
* `lib/adafruit_bus_device`
* `adafruite_register`
